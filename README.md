# Sistema de Ficheros — Biblioteca Universitaria Central

Minisitio interactivo para la exposición del proyecto final de Sistemas Operativos.

## Cómo abrir

Abre `index.html` directamente en el navegador, o sirve la carpeta con un servidor local:

```bash
# Python
python -m http.server 8080

# Node (npx)
npx serve .
```

Luego visita `http://localhost:8080`.

## Archivos

| Archivo      | Descripción                          |
|-------------|--------------------------------------|
| `index.html`| Estructura y contenido de las 8 secciones |
| `styles.css`| Diseño dashboard, diagramas CSS      |
| `app.js`    | Navegación, animaciones, contadores  |

## Tecnologías

HTML, CSS y JavaScript vanilla — sin frameworks.
